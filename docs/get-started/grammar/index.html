Getting Started With Sherpa

Well, no time like the present to get started! Check this out,

Run in Lab 
IGNORE { c:sp }

<> A > "hello" "world"
hello world

Cool huh, you’ll notice we are now in the Sherpa “lab”.

Note: these documents are integrated with Sherpa lab. Whenever you see the (TODO: Beaker symbol), you can click on it to open the lab to start experimenting with the related subject. Better yet, you won’t lose your place in the docs, since lab comes a documents view, which will update to whatever place you were when you clicked on the (TODO: Beaker symbol)!
